This folder contains all required scripts to run the sequencer.

These are:

<emailReport.R>           : The script required to send out the reports via email
<monsterlab_arduino.ino>  : The script that is executed on the Arduino
<generateImages.py>       : The python script for Craiyon API access
<monsterSequencer.Rmd>    : The RMarkdown script that orchestrates the whole process
<orchestrateSequencing.R> : Tool that orchestrates the whole process and takes the basic configuration. Run this to start the whole process
